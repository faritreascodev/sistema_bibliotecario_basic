Detalles del Proyecto

Explora la estructura y el funcionamiento del Sistema de Gestión Bibliotecaria.

Objetivo del Proyecto

Facilitar la consulta de libros a estudiantes mediante un sistema interactivo basado en semestre y materia.

Características Principales
  • Interfaz moderna y accesible.
  • Consulta rápida y eficiente.
  • Libros organizados por asignatura y semestre.
Mejoras Futuras
  • Integrar un backend robusto con autenticación de usuarios.
  • Implementar una base de datos para almacenar información de libros y usuarios.
  • Agregar funcionalidad de reservas de libros y notificaciones.
  • Optimizar la experiencia del usuario con sugerencias basadas en intereses.
